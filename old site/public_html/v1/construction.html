Construction Process

Member Fabrication

Firstly, each member must be fabricated as required, while meeting the constraints on member sizes given in the SDC Rules for each year. We hope to get the members laser cut for precision and time saving, but typically a jig is used to cut each member to size and cutout the needed slots for connections. This is probably the most time consuming test as each member needs to be measured, cut, checked and sanded before moving onto the next stage

Floor Fabrication

Next, with the members cut, we use floor templates to assemble the desired layout for the year. Once members are placed accordingly, we can glue the floor assembly together to let it dry. Then, vertical members can be glue on top to create a floor module. Before any vertical members are placed however, the floor plan is checked to ensure it is level. After the vertical members are added, again levelness is checked.

Module Fabrication

With each individual floor constructed, we assemble sets of 4-5 floors together into a module. Each module is checked for levelness and workmanship.

Tower Fabrication

With approximately 25 floors, we have between 5-6 modules for the tower. This step involves attaching one module at a time to the next to build the tower up. At each stage levelness and workmanship is checked.

Base Plate Installation

Finally, the base plate can be installed. This base plate is crucial as it is the piece responsible for coupling the shake table motion to the base of the structure. The columns are sanded down to ensure the entire structure is level and meets the specifications. Then, the bottoms of the columns are secured to the base plate.

Shipping the Tower

Before we can compete, we must create an appropriate shipping box for the tower. As you can imagine the tower is quite fragile and requires a unique box for shipping.
